# ⏳ 모래 가루 (SandPowder)
`모래 가루`는 파이썬 서버 라이브러리입니다.

## 제작 목적
모래 가루는 [`맑은한울`](https://github.com/clrhanul)의 서버 구축을 위한 목적으로 제작되었습니다.

## 목표
- [ ] 최소한 `HTTP 통신`은 되게 만들기
- [ ] `Routing`을 쉽고 간편하게 구현
- [ ] `SSL` `TLS` 암호화 기능 구현
- [ ] `HTTP/2` 구현
- [ ] `QUIC` 구현
- [ ] `HTTP/3` 구현

## 레포지토리 설명
### 브랜치 (Branch)
   브랜치  | 설명
:---------:| :----------
**`main`** | 배포용 브랜치
 **`dev`** | 개발용 브랜치. 배포 준비를 마친 경우 `main`으로 복제한다.
**`docs`** | 문서작성용 브랜치.
